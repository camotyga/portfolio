Graph Measures

This Python code implements graph structures and algorithms for analyzing graph traversal strategies. It defines classes for nodes, edges, and graphs, along with methods for adding nodes and edges. Additionally, it computes graph metrics such as centrality measures. The code includes an Agent class capable of performing random walks and finding shortest paths using the networkx library. Simulations compare random walk and shortest path strategies by measuring the average number of visited nodes. Overall, this project demonstrates proficiency in graph theory, algorithm design, and simulation techniques, making it suitable for showcasing expertise in data structures and network analysis.

Accessability Usability Report

This HTML code represents an individual accessibility report conducted on the HBO website. The report evaluates the website's compliance with web accessibility standards, specifically the Web Content Accessibility Guidelines (WCAG) 2.2. It provides detailed findings regarding positive aspects and areas requiring improvement in accessibility, supported by screenshots and analysis. Recommendations for enhancing accessibility are also included, aiming to improve usability for individuals with disabilities and ensure a more inclusive user experience. Overall, this report showcases proficiency in accessibility evaluation and emphasizes the importance of adhering to accessibility standards for web design.
